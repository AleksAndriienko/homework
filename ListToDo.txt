1.Cоздаю репозиторий на гитхаб под названием "homework"
2.Cоздаю в этом каталоге текстовый файл github
3.Создаю на локальном гите директорию /Git с помощью команды mkdir Git
4.Клонирую данный репозиторий из гитхаб на локальный с помощью команды:
   git clone https://github.com/AleksAndriienko/homework.git
5.Проверяю локальный каталог с помощью команды ls и вижу что в нашем каталоге /Git появилась 
  папка /homework c файлом github 
6.Добавляю новую ветку под названием second помощью команды git branch second
7.Что бы проверить, на какой ветке мы находимся - использую команду Git branch
8.На ветке second создаю файл под названием ListToDo.txt в директории /Git/homework c помощью
   команды touch ListToDo.txt  
   В итоге проверяю наш каталог с помощью команды ls  вижу что мы имеем  Git/homework github 
   ListToDo.txt
9. В фале ListToDo.txt записываю пошагово все операции домашнего задания.
10.Индексирую новый файл ListToDo.txt с помощью команды git add ListToDo.txt
11.Фиксирую изменения c кратким описанием с помощью команды git commit -m "file ListToDo.txt added"
12.Делаю заливку на гитхаб с помощью команды git push -u origin master:
    - ввожу логин и пароль
13.Проверю гитхаб и вижу что наши файлы github и ListToDo.txt находятся в директории мастер
    AleksAndriienko/homework
14.Возвращаюсь на локальный гит и делаю изменения в файле ListToDo.txt  , после чего повторно
   делаю шаги 10,11,12.
15.На локальном гит перехожу из ветки second в ветку master с помощью команды Git branch master 
16.В локальном гит обединяю директорию second c директорией master с помощью команды:
    - git merge second 

После чего получаем сообщение:
